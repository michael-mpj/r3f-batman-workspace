- **Node.js** >= 20.0.0

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/michael-mpj/r3f-batman-workspace.git
   cd r3f-batman-workspace
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start development:**

   ```bash
   npm run dev
   # or use Batman power
   npm run batman
   ```
